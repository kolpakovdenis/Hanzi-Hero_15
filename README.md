# Тренажёр ключей Канси (214 ключей)

## О проекте

Интерактивный тренажёр для изучения 214 ключей Канси — фундамента китайской письменности. Проект помогает запомнить начертания, произношение, значение и историю каждого ключа через визуализацию и практику.

## Функции

- **214 ключей** — полная система Канси с нумерацией
- **Анимация черт** — пошаговое отображение правильного порядка написания
- **Режим практики** — рисуйте ключи пальцем или мышью
- **Избранное** — сохраняйте сложные ключи для повторения
- **Озвучивание** — прослушивание произношения на китайском
- **Тёмная тема** — комфорт при любом освещении

## Технологии

- Hanzi Writer — рендеринг и анимация иероглифов
- LocalStorage — сохранение избранного и темы
- Адаптивная вёрстка — работает на любых устройствах

## Быстрый старт

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Начните изучение с первого ключа 一 (yī)
